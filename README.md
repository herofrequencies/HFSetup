# HFSetup

Public installers for Hero Frequencies applications. **No source code** — release binaries only.

## Downloads

Installers are published as [GitHub Releases](https://github.com/herofrequencies/HFSetup/releases) assets.

| Application | Installer |
|-------------|-----------|
| Preset Manager + mintserver | `HeroFrequenciesPresetManagerSetup.exe` |

## Preset Manager

Windows setup for **Hero Frequencies Preset Manager** and the **mintserver** device service (`HeroMintServer`).

Source code: private [Hero-Applications](https://github.com/herofrequencies/Hero-Applications) repository.
